Features

* External command executing with easy syntax
* Powerfull redirecting
* Basic Types

  * Nil
  * Bool
  * Int
  * Float
  * String
  * Array
  * Dictionary (Hash)
  * Function
  * UserFunction
  * Method

* Structured programming with ``if``, ``while`` and ``every`` statements
* Arithmetic operations
* Function
* UTF-8 (only) acceptable
* Written in Objective Caml
